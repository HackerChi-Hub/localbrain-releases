# LocalBrain

> One-click local AI for Apple Silicon Mac. Privacy-first local LLM manager.

LocalBrain turns your Mac into a private AI box — download, run, and integrate local MLX models with one click, fully offline.

**Website**: [hyphentech.top/localbrain](https://hyphentech.top/localbrain)

This repository hosts **public installer downloads only**. Source code is private.

---

## Download

Get the latest DMG from [Releases](https://github.com/HackerChi-Hub/localbrain-releases/releases/latest), or `LocalBrain-latest.dmg` in this repo.

Requirements: macOS on Apple Silicon (M1 or newer).

## Quick Start

1. Download the latest DMG
2. Drag `LocalBrain.app` to `/Applications`
3. Launch — it auto-detects your Mac and recommends the best model for your RAM
4. Download a model → click "启动" (Start) → chat locally

## Features

- Local MLX model manager with hardware-aware recommendations
- OpenAI-compatible local server (`http://127.0.0.1:11434/v1`)
- One-click integration with Codex, OpenCode, Claude Code, and ScreenLex
- Built-in multimodal MCP tools (Whisper transcription, TTS, image/video generation)
- Self-contained runtime — no system Python/dependencies required
